- Postman
- Jmeter
- CheckLists
- Bag Reports
- SQL
- Charles
- Mobile testing
=== каждую ветку создаем командой git branch name_branch
$ git branch Postman
$ git branch Jmeter
$ git branch CheckLists
$ git branch Bag_Reports (с пробелом нельзя)
$ git branch SQL
$ git branch Charles
$ git branch Mobile_testing (с пробелом нельзя)
